Dual File Structure Issue

This repository has files in TWO locations due to historical restructuring:

  • Root directory: src/, index.html, package.json, etc.
  • Web directory: web/src/, web/index.html, web/package.json, etc.

üîí Strict Rules for AI Assistants

NEVER DO THESE COMMANDS:

# These commands can destroy the local web/ folder:

git checkout <commit> -- index.html

SAFE OPERATIONS ONLY:

# Safe git status checking:

git log --oneline

# Safe directory operations:
