Gamification System Implementation Plan

Introduction

This document outlines the implementation of a gamification system for our Track & Field app. The goal is to introduce a Points & Badges system with social visibility to drive engagement without over-engineering.

Requirements

The original requirements from the user were:

We will be creating a new component for Gamification of our app: Introduce a Points & Badges system with social visibilityâ€”this drives engagement without over-engineering.

Core Concepts

Points

  • Award for every workout session logged, eating entry, or sleep record
  • Bonus for streaks (e.g. +10 points after 7 consecutive days)

Badges & Levels

  • Predefined badges (e.g. "First 5 Workouts", "7-Day Streak", "1000 Calories Tracked")
  • Levels that unlock based on total points

Streak Counter

  • Tracks consecutive days with at least one entry in each category (workout, nutrition, sleep)

Leaderboard

  • Overall and team leaderboards showing top point-earners

Component Structure

The gamification system will consist of the following components:

  • <PointsDisplay athleteId /> - Shows current points, level, and progress bar to next level
  • <BadgeList athleteId /> - Renders earned badges with icons and dates awarded
  • <StreakTracker athleteId /> - Displays current streak count and best-ever streak
  • <Leaderboard teamId /> - Shows top 5 athletes by points

Implementation Steps Overview

Points Logic

  • On each CRUD action (workout/sleep/eating), call a service to add points
  • After insertion, check for streaks or milestone badges

Badge Assignment

  • Maintain a rules engine (e.g. simple functions) that, given an athlete's history, returns new badges to award

Components & Hooks

  • usePoints(athleteId) returns total points and a function to fetch history
  • useBadges(athleteId) returns earned badges
  • useStreak(athleteId) computes current streak

UI

  • Create the four components (PointsDisplay, BadgeList, StreakTracker, Leaderboard) in a new folder src/features/gamification
