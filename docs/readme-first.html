🚀 Development Workflow

Getting Started:

# 1. Ensure Node.js 20+ is active
nvm use 20

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev

# 4. Open browser to http://localhost:5173

Available Scripts:

# Development
npm run dev          # Start dev server on port 5173
npm run build        # Build for production
npm run preview      # Preview production build

# Testing
npm run test         # Run tests
npm run test:watch   # Run tests in watch mode

# Database
npm run migrate:track-meets    # Run track meets migration
npm run check:track-meets      # Check track meets tables

🛡️ Safety Guidelines for AI Assistants

⚠️ Critical Safety Rules:

  • 🚨 NEVER restore files from git without explicit user permission
  • 🚨 NEVER run git checkout commands that could overwrite local changes
  • 🚨 NEVER assume file locations - always verify with list_dir first
  • 🚨 ASK PERMISSION before any git operations that modify files
  • 🚨 PRESERVE existing structure - don't move files between directories

✅ Safe Operations Protocol:

  1. Check current state: Use git status and list_dir first
  2. Edit existing files only: Use edit_file and search_replace
  3. Verify before git commands: Ask user permission for any git operations
  4. Focus on code changes: Modify content, not file structure
  5. Test locally first: Always build and test before deploying

🚫 Commands to AVOID:

# These can destroy local work:
git checkout <commit> -- src/
git checkout <commit> -- index.html
git reset --hard <commit>
git clean -fd

# Always ask permission before:
git restore <file>
git checkout <branch>
mv src/ web/src/
rm -rf <directory>
